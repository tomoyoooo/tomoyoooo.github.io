十大排序总结

算法 排序
算法
发布日期:   2023-02-26

排序算法总结

一、冒泡排序

  1. 思路：通过两两交换，每次外循环都把最大or最小的数放到最后，像水底的泡泡一样，越往上越大

  2. 时间复杂度：稳定，最好O(n)，最差O(n^2)，平均O(n^2)

  3. 空间复杂度：O(1)

  4. 代码示例

    public int[] BubbleSort(int[] arr){
    //每次循环把最大/最小的数放到了最后
    for(int i = 1; i < arr.length; ++i){
        boolean flag = true;//如果flag为true，代表本次循环无交换操作，排序结束
        for(int j = 0; j < arr.length-i; ++j){
            //把大的数往后移
            if(arr[j] > arr[j+1]){
                int tmp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = tmp;
                flag = false;
            }
        }
        if(flag) break;
    }
    return arr;
}

二、选择排序

  1. 思路：每次外循环找到最大or最小的数，直接放到最前面or最后面

  2. 时间复杂度：不稳定，最好O(n^2)，最差O(n^2)，平均O(n^2)

  3. 空间复杂度：O(1)

  4. 代码示例

    public int[] SelectionSort(int[] arr){
    for(int i = 0; i < arr.length; ++i){
        int minIndex = i;
        //找到后面最小的数
        for(int j = i+1; j < arr.length; ++j){
            if(arr[minIndex] > arr[j]) minIndex = j;
        }
        //把最小的数和当前位置的数交换，然后转到下一位置
        if(minIndex != i){
            int tmp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = tmp;
        }
    }
    return arr;
}

三、插入排序

  1. 思路：在数组前面构建一个有序区域，遍历到每一个数字都把它们通过两两交换插入到自己应该在的位置

  2. 时间复杂度：稳定，最好O(n)，最差O(n^2)，平均O(n^2)

  3. 空间复杂度：O(1)

  4. 代码示例

    public int[] InsertionSort(int[] arr){
    //i为当前需要插入的数
    for(int i = 1; i < arr.length; ++i){
        int preIndex = i-1;
        int current = arr[i];
        //一直把有序区域的数往后移，直到找到比当前插入数字小的数
        while(preIndex>=0 && current<arr[preIndex]){
            arr[preIndex+1] = arr[preIndex];
            preIndex--;
        }
        //插入当前数字
        arr[preIndex+1] = current;
    }
    return arr;
}

四、希尔排序

  1. 思路：也是一种插入排序，通过增加分组过程，减少了插入排序移动的次数，达到了O(nlogn)的时间复杂度。

  2. 时间复杂度：稳定，最好O(nlogn)，最差O(n^2)，平均O(nlogn)

  3. 空间复杂度：O(1)

  4. 代码示例

    public int[] ShellSort(int[] arr){
    int gap = arr.length/2;
    while(gap > 0){
        //把每组组内进行插入排序
        for(int i = gap; i < arr.length; ++i){
            int cur = arr[i];
            int preIndex = i-gap;
            while(preIndex>=0 && arr[preIndex]>cur){
                arr[preIndex+gap] = arr[preIndex];
                preIndex -= gap;
            }
            arr[preIndex+gap] = cur;
        }
        //减少组的数量，此时组内再进行交换的话，由于一半的数已经是有序的了，插入排序移动的次数会减少。
        gap /= 2;
    }
    return arr;
}

五、归并排序

  1. 思路：是一种分治算法，本质上类似于二叉树的后序遍历，先完成左右两部分的排序，再完成自身整体的排序。

  2. 时间复杂度：稳定，最好O(nlogn)，最差O(nlogn)，平均O(nlogn)

  3. 空间复杂度：O(n)

  4. 代码示例

    public int[] MergeSort(int[] arr){
    if(arr.length <= 1) return arr;
    int mid = arr.length/2;
    //分成左右两个部分
    int[] left = Arrays.copyOfRange(arr, 0, mid);
    int[] right = Arrays.copyOfRange(arr, mid, arr.length);
    //先递归完成左右两部分的排序，最后对自己排序
    return merge(MergeSort(left), MergeSort(right));
}
public int[] merge(int[] left, int[] right){
    int[] res = new int[left.length+right.length];
    int i = 0, j = 0;
    int idx = 0;
    //由于左右两部分是排好序的，只需要用双指针方法进行合并
    for(int k = 0; k < res.length; ++k){
        if(i==left.length){
            res[k] = right[j++];
        }
        else if(j==right.length){
            res[k] = left[i++];
        }
        else if(left[i] < right[j]){
            res[k] = left[i++];
        }
        else{
            res[k] = right[j++];
        }
    }
    return res;
}

六、快速排序

  1. 思路：也是一种分治算法，类似于二叉树的前序遍历，先通过一个基准数分区(为基准数找到自己在整体中应该在的位置，也就是前面的数都比自己小，后面的数都比自己大)，再对分成的左右两部分分别进行排序。

  2. 时间复杂度：不稳定，最好O(nlogn)，最差O(nlogn)，平均O(nlogn)

  3. 空间复杂度：O(nlogn)

  4. 代码示例

    public int[] QuickSort(int[] arr, int low, int high){
    if(low < high){
        //找到基准数在的位置
        int position = partition(arr, low, high);
        QuickSort(arr, low, position-1);
        QuickSort(arr, position+1, high);
    }
    return arr;
}
public int partition(int[] arr, int low, int high){
    int pivot = arr[high];
    int pointer = low;
    //把比基准数小的数全部放在前面
    for(int i = low; i < high; ++i){
        if(arr[i] <= pivot){
            int tmp = arr[i];
            arr[i] = arr[pointer];
            arr[pointer] = tmp;
            pointer++;
        }
    }
    //把基准数放到对应位置，此时前面的数全部小于基准数，后面的数全部大于基准数
    int tmp = arr[pointer];
    arr[pointer] = arr[high];
    arr[high] = tmp;
    return pointer;
}

七、堆排序

  1. 思路：也是构建有序区的思路，但是使用的是堆的数据结构来实现的，每次把最大or最小的数放到堆顶，然后和有序区当前点进行交换，有序区当前点往前移动一位。

  2. 时间复杂度：不稳定，最好O(nlogn)，最差O(nlogn)，平均O(nlogn)

  3. 空间复杂度：O(1)

  4. 代码示例

    int heapLen;
public int[] HeapSort(int[] arr){
    heapLen = arr.length;
    //建立最大堆
    buildMaxHeap(arr);
    //把堆顶与有序区当前判断点交换，有序区(i到arr.length-1)长度+1，堆对应长度heapLen-1
    for(int i = arr.length-1; i > 0; --i){
        swap(arr, 0, i);
        heapLen -= 1;
        heapify(arr, 0);
    }
    return arr;
}
public void swap(int[] arr, int i, int j){
    int tmp = arr[i];
    arr[i] = arr[j];
    arr[j] = tmp;
}
//维护最大堆，只要堆下面有更大的数，就会被传上来
public void heapify(int[] arr, int i){
    int left = 2*i+1;
    int right = 2*i+2;
    int largest = i;
    if(right<heapLen && arr[right]>arr[largest]) largest = right;
    if(left<heapLen && arr[left]>arr[largest]) largest = left;
    if(largest!=i){
        swap(arr, largest, i);
        //递归处理交换后的子堆
        heapify(arr, largest);
    }
}
public void buildMaxHeap(int[] arr){
    for(int i = arr.length/2-1; i>=0; --i){
        heapify(arr, i);
    }
}

八、计数排序

  1. 思路：通过找到最大值和最小值，创建一个新数组，索引为原数-最小值的差，然后依次拿出即可得到排好序的数组

  2. 时间复杂度：稳定，最好O(n+k)，最差O(n+k)，平均O(n+k)，k为最大最小值的差

  3. 空间复杂度：O(k)

  4. 代码示例

    public int[] getMinAndMax(int[] arr){
    int maxx = arr[0];
    int minn = arr[0];
    for(int i = 0; i < arr.length; ++i){
        if(arr[i] < minn) minn = arr[i];
        if(arr[i] > maxx) maxx = arr[i];
    }
    return new int[]{minn, maxx};
}
public int[] CountingSort(int[] arr){
    if(arr.length < 2) return arr;
    //找到最大最小值
    int[] extremum = getMinAndMax(arr);
    int minn = extremum[0];
    int maxx = extremum[1];
    //创建新数组
    int[] countArr = new int[maxx-minn+1];
    int[] res = new int[arr.length];
	//把所有数放进新数组中
    for(int i = 0; i < arr.length; ++i) countArr[arr[i]-minn]++;
    //算上自己前面一共有多少个数
    for(int i = 1; i < countArr.length; ++i) countArr[i] += countArr[i-1];
    //依次拿出，得到有序数组
    for(int i = arr.length-1; i>=0; --i){
        int idx = countArr[arr[i]-minn]-1;
        res[idx] = arr[i];
        countArr[arr[i]-minn]--;
    }
    return res;
}

九、桶排序

  1. 思路：通过一定方法进行分组(数字大小和分组需要有关)，分一次组后，前面组内的所有数字一定比后面组内的所有数字小or大，然后递归排序即可。

  2. 时间复杂度：稳定，最好O(n+k)，最差O(n^2)，平均O(n+k)，k为初始桶数

  3. 空间复杂度：O(k)

  4. 代码示例

    public int[] getMinAndMax(List<Integer> arr){
    int maxx = arr.get(0);
    int minn = arr.get(0);
    for(int i : arr){
        if(i < minn) minn = i;
        if(i > maxx) maxx = i;
    }
    return new int[]{minn, maxx};
}
public List<Integer> BucketSort(List<Integer> arr, int bucketSize){
    if(arr.size()<2 || bucketSize==0) return arr;
    int[] extremum = getMinAndMax(arr);
    int minn = extremum[0];
    int maxx = extremum[1];
    int bucketNum = (maxx-minn)/bucketSize+1;
    List<List<Integer>> buckets = new ArrayList<>();
    for(int i = 0; i < bucketNum; ++i){
        buckets.add(new ArrayList<Integer>());
    }
	//数字越大，会排到越后面的组
    for(int element : arr){
        int idx = (element-minn)/bucketSize;
        buckets.get(idx).add(element);
    }
    //递归使用桶排序
    for(int i = 0; i < buckets.size(); ++i){
        if(buckets.get(i).size()>1){
            buckets.set(i, BucketSort(buckets.get(i), bucketSize/2));
        }
    }
    //获取结果
    ArrayList<Integer> res = new ArrayList<>();
    for(List<Integer> bucket : buckets){
        res.addAll(bucket);
    }
    return res;
}

十、基数排序

  1. 思路：依次对每个数字的个位、十位、百位……进行排序，越到后面优先级越高

  2. 时间复杂度：稳定，最好O(nk)，最差O(nk)，平均O(n*k)，k是最大值的位数

  3. 空间复杂度：O(n+k)

  4. 代码示例

    public int[] RadixSort(int[] arr){
    if(arr.length<2) return arr;
    int N = 1;
    int maxx = arr[0];
    for(int i : arr) maxx = Math.max(maxx, i);
    //算出最大值的位数
    while(maxx/10 != 0){
        maxx /= 10;
        N++;
    }
    //对每一位进行排序，优先级越到后面越高，所以先从个位开始
    for(int i = 0; i < N; ++i){
        List<List<Integer>> radix = new ArrayList<>();
        for(int k = 0; k < 10; ++k){
            radix.add(new ArrayList<Integer>());
        }
        for(int element : arr){
            int idx = (element/(int)Math.pow(10,i))%10;
            radix.get(idx).add(element);
        }
        int idx = 0;
        //先放当前对应位数字小的
        for(List<Integer> l : radix){
            for(int n : l){
                arr[idx++] = n;
            }
        }
    }
    return arr;
}
